# Notes App (Full-Stack MERN)

A full-stack Notes application with authentication, CRUD operations, and search functionality.  
Built using **React, Node.js, Express, MongoDB**, and deployed on **Render**.

---

##  Features

###  Authentication
- User Signup & Login (Email + Password)
- Google OAuth Login
- JWT-based authentication
- Logout functionality

###  Notes Management (CRUD)
- Create new notes
- View all notes
- Edit existing notes
- Delete notes
- Notes are user-specific

###  Search
- Real-time note filtering by title or content

###  Security
- Password hashing using bcrypt
- Protected routes using auth middleware

---

##  Tech Stack

### Frontend
- React
- React Router
- CSS

### Backend
- Node.js
- Express.js
- MongoDB + Mongoose
- JWT (JSON Web Tokens)
- bcrypt
- cookie-parser
- CORS
